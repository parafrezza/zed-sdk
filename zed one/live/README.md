# ZED SDK - ZED One Live Capture

This sample shows how to capture and display live video from a ZED One camera.

## Features

- Open and initialize a ZED One camera
- Capture frames in real-time
- Display video using OpenCV
- Print camera information (model, resolution, FPS, serial number)

## Usage

### C++

```bash
cd build
./ZED_One_Live
```

### Python

```bash
python live.py
```

The sample will open the camera, display live video, and print camera information. Press 'q' to quit.
